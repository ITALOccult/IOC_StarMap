# Elementi UI (Titolo, Bordo, Bussola, Scala)

Configurazione degli elementi dell'interfaccia sulla mappa.

## Titolo

```json
{
  "title": {
    "enabled": true,
    "text": "My Star Map",
    "position": 1,
    "custom_x": 0.0,
    "custom_y": 0.0,
    "font_size": 18.0,
    "text_color": 4294967295,
    "background_color": 2147483903,
    "show_background": true,
    "padding_x": 10.0,
    "padding_y": 10.0
  }
}
```

### Parametri

| Campo | Tipo | Default | Descrizione |
|-------|------|---------|-------------|
| `enabled` | bool | true | Mostra titolo |
| `text` | string | "" | Testo del titolo |
| `position` | int | 1 | Posizione (0-5) |
| `custom_x` | float | 0.0 | X custom (0-1) se position=5 |
| `custom_y` | float | 0.0 | Y custom (0-1) se position=5 |
| `font_size` | float | 18.0 | Dimensione font |
| `text_color` | uint32 | 0xFFFFFFFF | Colore testo |
| `background_color` | uint32 | 0x00000000 | Colore sfondo |
| `show_background` | bool | false | Mostra sfondo |
| `padding_x` | float | 10.0 | Padding orizzontale |
| `padding_y` | float | 10.0 | Padding verticale |

### Posizioni

| Valore | Nome | Posizione |
|--------|------|-----------|
| 0 | NONE | Disabilitato |
| 1 | TOP_LEFT | Alto sinistra |
| 2 | TOP_RIGHT | Alto destra |
| 3 | BOTTOM_LEFT | Basso sinistra |
| 4 | BOTTOM_RIGHT | Basso destra |
| 5 | CUSTOM | Personalizzata (usa custom_x, custom_y) |

### Esempi

**Top-left standard**:
```json
{
  "title": {
    "text": "Orion Region",
    "position": 1,
    "font_size": 20.0
  }
}
```

**Custom posizionamento (centro-destra)**:
```json
{
  "title": {
    "text": "M42",
    "position": 5,
    "custom_x": 0.85,
    "custom_y": 0.5,
    "font_size": 24.0,
    "show_background": true
  }
}
```

## Bordo

```json
{
  "border": {
    "enabled": true,
    "color": 4294967295,
    "width": 2.0,
    "margin": 0.0
  }
}
```

### Parametri

| Campo | Tipo | Default | Descrizione |
|-------|------|---------|-------------|
| `enabled` | bool | true | Mostra bordo |
| `color` | uint32 | 0xFFFFFFFF | Colore bordo (bianco) |
| `width` | float | 2.0 | Spessore in pixel |
| `margin` | float | 0.0 | Margine interno in pixel |

### Esempi

**Bordo sottile**:
```json
{
  "border": {
    "width": 1.0,
    "color": 3435973887,
    "margin": 5.0
  }
}
```

**Bordo spesso**:
```json
{
  "border": {
    "width": 4.0,
    "color": 255,
    "margin": 10.0
  }
}
```

**Disabilitato**:
```json
{
  "border": {
    "enabled": false
  }
}
```

## Bussola

```json
{
  "compass": {
    "enabled": true,
    "position": 2,
    "custom_x": 0.0,
    "custom_y": 0.0,
    "size": 80.0,
    "north_color": 4278190335,
    "east_color": 16711935,
    "line_color": 4294967295,
    "line_width": 2.0,
    "show_labels": true,
    "label_font_size": 10.0
  }
}
```

### Parametri

| Campo | Tipo | Default | Descrizione |
|-------|------|---------|-------------|
| `enabled` | bool | true | Mostra bussola |
| `position` | int | 2 | Posizione (0-5) |
| `custom_x` | float | 0.0 | X custom (0-1) |
| `custom_y` | float | 0.0 | Y custom (0-1) |
| `size` | float | 80.0 | Dimensione in pixel |
| `north_color` | uint32 | 0xFF0000FF | Colore Nord (rosso) |
| `east_color` | uint32 | 0x00FF00FF | Colore Est (verde) |
| `line_color` | uint32 | 0xFFFFFFFF | Colore linee (bianco) |
| `line_width` | float | 2.0 | Spessore linee |
| `show_labels` | bool | true | Mostra N/E/S/W |
| `label_font_size` | float | 10.0 | Dimensione font |

### Posizioni

Stesse posizioni di titolo (0-5):
1 = TOP_LEFT, 2 = TOP_RIGHT, 3 = BOTTOM_LEFT, 4 = BOTTOM_RIGHT, 5 = CUSTOM

### Esempi

**Standard (top-right)**:
```json
{
  "compass": {
    "position": 2,
    "size": 100.0,
    "show_labels": true
  }
}
```

**Custom (bottom-left)**:
```json
{
  "compass": {
    "position": 3,
    "size": 120.0,
    "north_color": 4278190335,
    "east_color": 4294967295,
    "line_width": 3.0
  }
}
```

**Disabilitata**:
```json
{
  "compass": {
    "enabled": false
  }
}
```

## Scala Angolare

```json
{
  "scale": {
    "enabled": true,
    "position": 3,
    "custom_x": 0.0,
    "custom_y": 0.0,
    "length": 100.0,
    "color": 4294967295,
    "line_width": 2.0,
    "font_size": 10.0,
    "show_text": true
  }
}
```

### Parametri

| Campo | Tipo | Default | Descrizione |
|-------|------|---------|-------------|
| `enabled` | bool | true | Mostra scala |
| `position` | int | 3 | Posizione (0-5) |
| `custom_x` | float | 0.0 | X custom (0-1) |
| `custom_y` | float | 0.0 | Y custom (0-1) |
| `length` | float | 100.0 | Lunghezza in pixel |
| `color` | uint32 | 0xFFFFFFFF | Colore (bianco) |
| `line_width` | float | 2.0 | Spessore linea |
| `font_size` | float | 10.0 | Dimensione font |
| `show_text` | bool | true | Mostra valore in gradi |

### Conversione Lunghezza

La libreria calcola automaticamente il valore angolare:
```
angle_deg = length_px × FOV_deg / image_width_px
```

Esempio: FOV 10°, immagine 2400px, length 120px
```
angle = 120 × 10 / 2400 = 0.5°
```

### Esempi

**Standard (bottom-left)**:
```json
{
  "scale": {
    "position": 3,
    "length": 100.0,
    "show_text": true
  }
}
```

**Piccolo e sottile**:
```json
{
  "scale": {
    "position": 4,
    "length": 50.0,
    "line_width": 1.0,
    "font_size": 8.0
  }
}
```

**Disabilitata**:
```json
{
  "scale": {
    "enabled": false
  }
}
```
